# 🍽️ CalorieSight クイックスタートガイド

プログラミング経験が浅くても大丈夫！
この手順通りに進めれば、アプリが動きます。

---

## ステップ1: Node.js をインストール

すでにインストール済みなら飛ばしてOK。

1. https://nodejs.org/ にアクセス
2. 「LTS（推奨版）」をダウンロード
3. インストーラーを実行（全てデフォルトでOK）
4. 確認: コマンドプロンプト（またはターミナル）で以下を実行
   ```
   node --version
   ```
   → v20.x.x のような数字が出ればOK

---

## ステップ2: プロジェクトを準備

1. このZIPを好きな場所に解凍
2. コマンドプロンプト（またはターミナル）を開く
3. 解凍したフォルダに移動
   ```
   cd C:\Users\あなたの名前\Desktop\caloriesight
   ```
   ※パスは実際の場所に変えてください

4. 依存パッケージをインストール
   ```
   npm install
   ```
   → 数分かかります。エラーが出ても「WARN」は無視してOK

---

## ステップ3: APIキーを設定（任意）

**APIキーなしでもデモモードで動きます！**
まずはAPIキーなしで試して、後から設定してもOKです。

### 設定する場合:

1. `.env.example` ファイルをコピーして `.env` にリネーム
2. `.env` をテキストエディタで開く
3. APIキーを記入:

```
VITE_GEMINI_API_KEY=ここにGeminiのAPIキー
VITE_CLAUDE_API_KEY=ここにClaudeのAPIキー
```

### APIキーの取得方法:

**Gemini API（食事写真認識用・無料枠あり）:**
1. https://aistudio.google.com/apikey にアクセス
2. Googleアカウントでログイン
3. 「APIキーを作成」をクリック
4. 表示されたキーをコピー

**Claude API（AIコーチング用・従量課金）:**
1. https://console.anthropic.com/ にアクセス
2. アカウント作成（クレジットカード登録必要）
3. 「API Keys」→「Create Key」
4. 表示されたキーをコピー

---

## ステップ4: 開発サーバーを起動

```
npm run dev
```

表示されるURL（例: http://localhost:5173/caloriesight/）を
ブラウザで開きます。

**これで完了！アプリが表示されます 🎉**

---

## ステップ5: 使い方

### 初回設定
1. ⚙️設定 タブで身長・年齢・体重・性別を入力
2. 目標体重と期限を設定
3. 「保存する」を押す → BMR・TDEE・目標が自動計算されます

### 毎日の使い方
1. 📸食事 → 食事を記録（写真AI認識 or 手動検索）
2. ⚖️体重 → 朝の体重を記録
3. 🏃運動 → 運動した内容を記録
4. 💧習慣 → 水分と睡眠を記録
5. 📊ホーム → 今日の収支をチェック！

---

## スマホでも使いたい場合

### 同じWi-Fiネットワーク内なら:
1. `npm run dev -- --host` で起動
2. 表示される `http://192.168.x.x:5173/caloriesight/` をスマホで開く
3. スマホのブラウザで「ホーム画面に追加」

### 公開する場合（GitHub Pages）:
1. GitHubにリポジトリを作成
2. `npm run deploy` を実行
3. https://あなたのユーザー名.github.io/caloriesight/ でアクセス可能に

---

## トラブルシューティング

### `npm install` でエラーが出る
→ Node.js のバージョンが古い可能性。v20以上を使ってください。

### 画面が真っ白
→ ブラウザの開発者ツール（F12）→ Console タブでエラーを確認

### API認識が動かない
→ .env ファイルのAPIキーを確認。なくてもデモモードで動きます。

### データを消したい
→ ブラウザの開発者ツール → Application → IndexedDB → CalorieSightDB を削除
